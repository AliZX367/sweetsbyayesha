# Sweets by Ayesha

Halal-certified home bakery near Schaumburg, IL specializing in cake pops, rice krispie treats, and custom baked goods.

## Getting started

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Pages

- `/` Home
- `/menu` Our Treats
- `/about` About Ayesha
- `/order` Place an Order (client-side form; logs to console and shows success)

## SEO

- `app/sitemap.ts` generates `/sitemap.xml`
- `app/robots.ts` generates `/robots.txt`
